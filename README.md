# extracting_self-similarity

This is the code and data accompanying the paper "Extracting self-similarity from data", by [Bempedelis, Magri, and Steiros, Journal of Fluid Mechanics, 1020, A11, (2025)](https://doi.org/10.1017/jfm.2025.10656). If you use this method and/or code, please cite the above work.
